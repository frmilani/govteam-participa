# Story E1.3: Renomear Aba "Empresas" → "Entidades" com Label Contextual

## Status
Done

## Story
**As a** administrador configurando uma enquete,
**I want** a aba "Empresas" no formulário de enquete exibir o nome correto baseado no tipo de pesquisa (Empresas, Candidatos, Serviços, Dimensões),
**so that** a nomenclatura seja intuitiva para qualquer tipo de pesquisa que eu esteja configurando.

## Acceptance Criteria

1. - [x] Aba renomeada de "Empresas" para label contextual no array `tabs` do EnqueteForm
2. - [x] Label contextual baseado no campo `tipoPesquisa` da enquete:
   - `premiacao` → "Empresas Participantes"
   - `politica` → "Candidatos"
   - `politicas-publicas` → "Serviços e Propostas"
   - `corporativo` → "Dimensões Avaliadas"
   - `custom` → "Entidades"
3. - [x] Enquetes existentes (sem `tipoPesquisa`) exibem label "Empresas Participantes" (backward compat)
4. - [x] Aba fica collapsed (opacity-40, cursor-pointer, hover:opacity-70) quando tipo não exige pré-cadastro
5. - [x] Admin pode expandir manualmente aba collapsed clicando (collapsedOverride state)
6. - [x] Transição suave ao trocar tipo (transition-all duration-300 em label e tab)
