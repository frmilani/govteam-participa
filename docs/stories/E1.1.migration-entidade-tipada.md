# Story E1.1: Migration Prisma — Entidade Tipada e Novos Modelos

## Status
Done

## Story
**As a** administrador do sistema,
**I want** o modelo de dados evoluído para suportar entidades além de "Empresa" (candidatos, propostas, marcas, dimensões, serviços públicos),
**so that** o sistema possa ser usado como Motor Universal de Pesquisa atendendo premiações, pesquisas políticas, políticas públicas e corporativas.

## Acceptance Criteria

1. - [x] Enum `TipoEntidade` criado com valores: `EMPRESA`, `CANDIDATO`, `PROPOSTA`, `MARCA`, `DIMENSAO`, `SERVICO_PUBLICO`, `OUTRO`
2. - [x] Campo `tipo TipoEntidade @default(EMPRESA)` adicionado ao modelo `Estabelecimento`
3. - [x] Campo `metadados Json?` adicionado ao modelo `Estabelecimento`
4. - [x] Índice `@@index([organizationId, tipo])` adicionado ao modelo `Estabelecimento`
5. - [x] Modelo `TemplateQualidade` criado (id, organizationId, nome, perguntas[], segmentos[], timestamps)
6. - [x] Modelo `PerguntaQualidade` criado com relação cascade para TemplateQualidade (id, templateQualidadeId, texto, tipo, obrigatorio, opcoes, ordem)
7. - [x] Modelo `RespostaQualidade` criado (id, respostaId, perguntaId, categoriaId, valor, criadoEm)
8. - [x] Modelo `VotoLivre` criado (id, respostaId, categoriaId, textoOriginal, chavesFoneticas, consolidadoEmId, criadoEm)
9. - [x] Campo `dadosDemograficos Json?` adicionado ao modelo `Lead`
10. - [x] Campos adicionados ao modelo `Enquete`: `tipoPesquisa String @default("premiacao")`, `modoColeta String @default("recall-assistido")`, `incluirQualidade Boolean @default(false)`, `modoDistribuicao String @default("grupo")`, `maxCategoriasPorEleitor Int?`, `randomizarOpcoes Boolean @default(true)`, `configPesquisa Json?`
11. - [x] Relações `respostasQualidade RespostaQualidade[]` e `votosLivres VotoLivre[]` adicionadas ao modelo `Resposta`
12. - [x] Campo `templateQualidadeId String?` e relação `templateQualidade TemplateQualidade?` adicionados ao modelo `Segmento`
13. - [x] Migration roda sem erros em banco com dados existentes (dados preservados, estabelecimentos recebem `tipo: EMPRESA`)
14. - [x] `npx prisma generate` executa com sucesso
