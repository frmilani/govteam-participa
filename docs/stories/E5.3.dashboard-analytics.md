# Story E5.3: Dashboard de Analytics no Admin

## Status
Done

## Story
**As a** administrador abrindo o painel da enquete encerrada,
**I want** ver gráficos interativos de ranking, distribuição demográfica e radar de qualidade numa tela única,
**so that** eu consiga demonstrar ao cliente os resultados completos da pesquisa em 10 minutos de reunião sem depender de ferramentas externas.

## Acceptance Criteria

1. - [x] Nova aba ou botão **"📊 Analytics"** acessível na página de detalhes/gestão da enquete (ex: `/admin/enquetes/[id]`).
2. - [x] **Seção Visão Geral**: Cards de métricas — Total Respostas, Taxa de Conclusão (%), Período ativo, Mix de tipos de votos (donut simples).
3. - [x] **Seção Ranking**: Tabela interativa das entidades mais votadas. Colunas: Posição (🥇🥈🥉 nas top 3), Entidade, Votos, % do Total. Filtro por Categoria (dropdown). Destacar top 3 com cores/badges.
4. - [x] **Seção Perfil Demográfico**: Agrupamento por campo demográfico disponível nos dados. Para cada campo: gráfico de barras horizontal (`recharts` ou `chart.js`). Filtro "Ver eleitores de qual entidade?" conectado com o Select de entidades para fazer o cross-drill.
5. - [x] **Seção Qualidade**: Radar chart por entidade (1 entidade por vez, seletor). Eixos = perguntas do template. Tabela abaixo com ranking médio de qualidade. Só aparece se `incluirQualidade` for true e houver dados.
6. - [x] Botão **Exportar CSV** — gera CSV de ranking consolidado (sem dados individuais). Usa `json2csv` ou similar.
7. - [x] Se qualquer seção for baseada em dados insuficientes (< 5 respostas), mostrar aviso `"Dados insuficientes para exibição"` em vez de gráfico.
8. - [x] Skeleton loaders enquanto os 4 fetches carregam (um por seção, em paralelo via `Promise.all`).
9. - [x] Layout Desktop-first; mínimo de responsividade para tablet.

## Dev Notes

### Importação dinâmica dos Charts (evita erro de SSR)
```typescript
const RadarChart = dynamic(() => import('recharts').then(m => m.RadarChart), { ssr: false });
```

### Paralelismo dos fetches
```typescript
const [overview, ranking, demographics] = await Promise.all([
  fetchOverview(id), fetchRanking(id), fetchDemographics(id)
]);
```

### Tabela de Ranking — top 3 badges
Usar variant do Badge com cores: `gold`, `silver`, `bronze` ou emojis 🥇🥈🥉 + `className` condicional via `cn()`.
